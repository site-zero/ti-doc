# 控件族类

![](/com_races.png)

  ID     |  NAME    | DESCRIPTION
---------|----------|-----------------------
`INPUT`  | **输入** | 生产比较简单单一的数据类型
`EIDT`   | **编辑** | 生产比较复杂的数据
`SHELF`  | **搁架** | 组合更多子控件，用作界面布局
`ACTION` | **动作** | 按钮、导航条、等执行用户命令
`TILE`   | **瓦片** | 小面积信息展示控件
`PLAY`   | **播放** |展示多媒体内容
`VIEW`   | **视图** | 通常来处理比较复杂的数据，在页面上占用的面积也比较大
`DESIGN` | **设计** |提供复杂的界面生产能力
